# java-explore-with-me

**Explore With Me** (англ. «исследуй со мной») — приложение которое позволит пользователям делиться информацией об интересных событиях и находить компанию для участия в них.

Проект состоит из двух основных модулей:

**Основной сервис (main-service)** - управление событиями, пользователями, категориями и запросами на участие
**Сервис статистики (stats-service)** - сбор и анализ статистики просмотров событий
